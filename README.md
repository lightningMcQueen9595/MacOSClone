## 📌 What Is This?

**macOS Desktop Clone** is a front-end engineering project that faithfully recreates the look, feel, and interactions of Apple's macOS desktop environment — right inside a web browser.

No frameworks. No libraries. Just clean, handcrafted **HTML**, **CSS**, and **JavaScript**.

This project was built to sharpen UI/UX skills, understand the complexity behind desktop-like experiences, and demonstrate how powerful the modern web platform truly is.

<br/>

## ✨ Features at a Glance

| Feature | Description |
|---|---|
| 🖥️ **Realistic Desktop** | Full-screen desktop UI that mirrors the macOS environment |
| 📌 **Animated Dock** | Hover magnification effect just like the real macOS dock |
| 📂 **App Icons** | Clickable, interactive application icons on the desktop |
| 🪟 **Window Components** | Draggable/stylized window-style UI panels |
| 🎨 **Pixel-Perfect Design** | Every element carefully styled to match macOS aesthetics |
| ⚡ **Smooth Animations** | CSS-powered transitions for a fluid, native-feeling experience |

## ⚙️ Quick Setup

Get it running locally in under 60 seconds:

```bash
# 1. Clone the repository
git clone https://github.com/TanviNema1/MacOS-Clone-.git

# 2. Navigate into the folder
cd MacOS-Clone-

# 3. Open in browser
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

> ✅ No installations. No dependencies. Just open and run.

## 🧠 What I Learned

Working on this project built strong understanding of:

- **UI/UX Replication** — Analyzing and reconstructing a complex real-world interface
- **CSS Animations** — Building the dock magnification effect with pure CSS transforms
- **DOM Manipulation** — Handling clicks, hovers, and window interactions in vanilla JS
- **Component Thinking** — Breaking down a desktop UI into reusable, modular parts
- **Front-End Architecture** — Keeping HTML, CSS, and JS clean and well-organized at scale
